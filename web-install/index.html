Web Install Sample

A debugging tool for the Web Install API. See more info in the explainer and to-do's in this GitHub issue.

Intended use cases

0 param, current doc, same origin (with manifest id)

Executes:


      navigator.install();

1 param, background doc, cross origin (with manifest id)

Executes:


      let install_url = "https://mustjab.github.io/";
      navigator.install(install_url);

2 param, background doc, cross origin

Executes:


      let install_url = "https://diek.us/bubble/";
      let manifest_id = "https://diek.us/bubble/index.html";
      navigator.install(install_url, manifest_id);

2 param, background doc, cross origin, with screenshots

Executes:


      let install_url = "https://squoosh.app/";
      let manifest_id = "https://squoosh.app/?utm_medium=PWA&utm_source=launcher";
      navigator.install(install_url, manifest_id);
