Web Install Sample

A debugging tool for the Web Install API. See more info in the explainer and to-do's in this GitHub issue.

Install cross-origin with preset id

Executes:


      let manifest_id = "https://diek.us/bubble/index.html";
      let install_url = "https://diek.us/bubble/";
      navigator.install(manifest_id, install_url);

Install same-origin with two parameters

Executes:


      let manifest_id = "https://kbhlee2121.github.io/pwa/web-install/";
      let install_url = "https://kbhlee2121.github.io/pwa/web-install/index.html";
      navigator.install(manifest_id, install_url);

Install same-origin with zero parameters

Executes:


      navigator.install();
